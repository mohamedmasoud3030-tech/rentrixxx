# Rentrix Documentation Archive

This directory contains historical Rentrix reports, audits, evidence packs, and superseded indexes.

Archived docs are reference-only. Do not treat them as current roadmap, policy, product scope, release status, migration status, Supabase evidence, or Vercel evidence unless an active doc explicitly promotes the same fact and current code or migrations verify it.

Read current guidance first:

1. `../../AGENTS.md`
2. `../../README.md`
3. `../ai/CURRENT_EXECUTION_CONTEXT.md`
4. `../RENTRIX_MASTER_PLAN.md`
5. `../FIRST_CLIENT_DELIVERY_PLAN.md`
6. `../ai/ONBOARDING.md`

Archived material is grouped by its original purpose:

- `ai/` - superseded AI guidance, audits, product-scope notes, and indexes.
- `audits/` - dated schema, migration, and integrity audits.
- `demo-gates/` - historical demo-readiness evidence packs.
- `decisions/` - superseded ADR files retained for decision history.
- `reconciliation/` - historical repository comparison and cleanup inventories.
- `release-checkpoints/` - dated release handoff notes.
- `stabilization/` - historical stabilization evidence.
- `v01/` and `v02/` - dated release-thread status and review reports.
- `wave1/` - historical rollout reconciliation notes.
