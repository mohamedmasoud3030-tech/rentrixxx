### High-level status

- The current app is TanStack Router + React Query + Supabase wired through `AppProviders`, `AppRouterProvider`, `routeTree`, and `AppShell`.
- There is no forbidden current-app import from `legacy-src`, and the required forbidden search found no current usage of `legacy-src`, `useApp`, `AppContext`, `dataService`, `local db`, or `react-router-dom` in `artifacts/rentrix/src`.
- Core routed CRUD exists for properties, units-through-property-detail, people, contracts, financials, reports, maintenance, accounting, settings, login, and dashboard.
- The deepest disconnected area is reporting/financials: strong report services and hooks exist, but the `/reports` page bypasses them and recomputes local Supabase queries/charts.
- Arrears service/hook coverage exists in `financialReportsService` and `useFinancialReports`, but there is no dedicated surfaced current UI workflow for aged receivables, overdue invoices, or arrears summary.
- Accounting, settings, offline cache, owner/tenant relationship depth, WhatsApp/communications, and PDF/document generation are mostly shell/reference-only in current app while much richer legacy code exists as reference material.
- Recommendation: do **not** build new features yet. First wire the existing disconnected code in small PRs.
