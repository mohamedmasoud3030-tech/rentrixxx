# OpenManus Tools Integration
This directory contains tools integrated from the OpenManus project to enhance the development workflow of Rentrix.

## Integrated Tools:
1. **Planning Tool**: Advanced step-by-step planning and tracking for complex migrations and features.
2. **StrReplaceEditor**: Robust string-based file editing to avoid common errors in automated code modifications.

## Usage with Opus 4.5:
These tools are intended to be used by the Claude Opus 4.5 model with `effort: high` to ensure precise execution and high-quality results.
